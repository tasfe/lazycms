一、程序介绍
    1.LazyCMS v2.0 主要围绕简单、简洁、高效、自由，这个核心思想进行开发。
    2.理想结果：操作简单、界面简洁、执行高效、自定义性强
    3.界面更加人性化，完善的提示帮助气泡
    4.半面向对象、半面向过程的系统内核

二、实现功能
    1.人性化的引导安装程序
    2.支持MySQL、SQLite数据库
    3.用户管理，支持用户组，支持自定义用户资料表单
    4.支持用户组权限任意定制
    5.文件管理器，支持所有上传文件统一管理
    6.可视化修改网站的各种配置
    7.自由安装卸载系统模块，在线安装，上传安装
    8.自定义后台快捷方式，实现ie收藏夹的类似功能
    9.单页面管理、添加，实现分类显示
    10.文章分类无限分级，实现可记忆化的分类展开关闭功能
    11.栏目列表表单自定义，二级域名绑定
    12.文章模型自定义，增加表单值验证
    13.文章添加，自动推荐关键词，自动截取描述
    14.文章关键词自动内连，自动匹配相关文章
    15.系统支持多语言，多模板，可自由切换
    16.简化模板制作概念，XML风格的模板标签
    17.完善的菜单管理器，后台创建前台菜单
    18.安全中心，帮助用户提高网站的安全级别
